# Traffic Volume Prediction Using Machine Learning

This project focuses on predicting road traffic volume using Machine Learning techniques. The system analyzes historical traffic data along with environmental and time-based factors such as weather conditions, holidays, temperature, and timestamps to accurately forecast traffic congestion levels.

The project demonstrates a complete Machine Learning workflow, including:

* Data collection and preprocessing
* Handling missing and duplicate values
* Exploratory Data Analysis (EDA)
* Feature engineering and data encoding
* Model training and evaluation
* Traffic volume prediction using regression algorithms

The following regression models were trained and compared to achieve better prediction accuracy:

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Joblib

## Project Workflow

1. Import and preprocess the dataset
2. Perform data visualization and analysis
3. Split the dataset into training and testing sets
4. Train Machine Learning models
5. Evaluate models using metrics such as MSE and R² Score
6. Save trained models using Pickle/Joblib
7. Predict future traffic volume

## Objective

The main objective of this project is to develop an intelligent traffic prediction system that can support:

* Traffic management
* Congestion reduction
* Smart city planning
* Transportation optimization
